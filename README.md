写真のサイズは大きすぎると大変なので、以下のコードを<head>内の<title>の次に貼り付けて使うと写真のサイズがいい感じになるので使ってください
<style>
        .my-image {
            width: 320px;
            max-width: 100%;
            height: auto;
        }
    </style>
